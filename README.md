### 3. Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Sunny-W-Park/ensemble4publish
   ```
2. Install requirements
   ```sh
   pip3 install -r requirements.txt
   ```
3. Run
   ```sh
   python manage.py makemigrations
   python manage.py migrate
   python manage.py runserver
   ```
